# Hardware Planet 24 - Fullstack App

Fullstack-Anwendung für **Hardware Planet 24** (Gaming-Hardware & PC-Komponenten). Besteht aus einem C#-Backend und einem React-Frontend.

## 🚀 Tech Stack
* **Backend:** ASP.NET Core Web-API (C#) & Swagger UI
* **Frontend:** React (Vite), JavaScript Fetch API
